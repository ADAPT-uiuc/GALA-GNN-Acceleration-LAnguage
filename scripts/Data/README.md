# Steps

1. Activate the (dgl) environment
```commandline
conda activate dgl
```
2. Run the `get_all_datasets.py` file to download all the datasets.
```commandline
python get_all_datasets.py
```
3. Run the `get_all_datasets.py --wisegraph` file to download all the datasets for wisegraph.
```commandline
python get_all_datasets.py
```
